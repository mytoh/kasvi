kasvi
=====
build lehti app


Usage:
------
::

      $ kasvi leh yourapp
      $ tree yourapp
      yourapp/
      ├── lib
      │   ├── yourapp
      │   │   └── core.scm
      │   └── yourapp.scm
      ├── readme.rst
      └── yourapp.leh
      
      2 directories, 4 files


Install:
--------
if you have lehti installed
::

      lehti install kasvi
